# Fake-News-Stance-Detection
Headline-Body Stance Detection using Machine Learning<br><br>

This project builds a Machine Learning model to detect the stance relationship between a news headline and its corresponding article body.

The model classifies the relationship into one of the following categories:

Agree

Disagree

Discuss

Unrelated<br><br>
